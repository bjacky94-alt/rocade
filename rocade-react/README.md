# Rocade React

Migration JavaScript/React du prototype Flask.

## Stack

- Frontend: React + Vite
- Backend: Express
- Base locale: SQLite (fichier rocade.sqlite3)

## Installation

1. Se placer dans le dossier du projet:

   cd rocade-react

2. Installer les dependances:

   npm install

3. Lancer frontend + backend:

   npm run dev

## URLs

- Frontend: http://localhost:5173
- API: http://localhost:3001

## API

- GET /api/rocade/state
- PUT /api/rocade/state
- DELETE /api/rocade/state
